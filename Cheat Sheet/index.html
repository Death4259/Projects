HTML Basics

Tag Name Description
<!DOCTYPE_html> Doctype Declares document type, should always be the start to your code.
<html> HTML Your code should be contained within this tag
<head> Head This is where you put your page title (what appears on the tab), this is also where you link your CSS style sheet.
<body> Body This is where the "Meat" of your webpage is.
<table> Table You can use the table tag to contain and display data in a neat format.
